    2.  **For other 'intent's**: Confirm if ALL slots present in the input are filled.
    3.  **Otherwise**: Request the first null slot from the input.
    4.  **CRITICAL**: You MUST NOT invent any slot names. Use ONLY the slot names provided in the input JSON.

---

- Generate the response based on the action decided by the DM.
- Don't create too long responses.
- Don't add unnecessary information.
- Write only the response, without any additional text or explanation.

DM response: {action: confirmation, parameter: order_car}
NLU response: {intent: order_car, slots: {'car_id': car_id, 'price': price, 'name': user_name, 'surname': user_surname, 'id': id_number}}
database output: {
    "CarID": car_id, "brand": brand, "model": model, "year": year, "budget": budget, "Seats": seats, "car_type": car_type, "transmission": transmission
}

You have to provide a message that confirms the order of the car that matches the user's request.
